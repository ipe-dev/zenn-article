laravelで単体テストを実装していて、Exceptionが発生することをテストの仕方を忘れるのでメモ
# expectExceptionを使う
実装例

```php
public function testHogeException()
{
    $hoge = new HogeClass();

    $this->expectException() // 必ず先にexpectExceptionを実行

    $hoge->hogeFunction() //
}
```
